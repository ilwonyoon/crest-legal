# Crest Privacy Policy

Last updated: 2026-03-03

Crest ("we", "our", "us") provides a focus timer application.
This Privacy Policy explains what information we process and how it is used.

## 1. Information We Process

We process the following categories of data:

- App usage and settings stored on device:
  - Timer modes, durations, display preferences, and history records
- Optional location data:
  - If location permission is granted, location may be used for Together View map placement
  - Intended display scope is coarse context (for example, country/state-level context), not precise address display

We do not use your data for advertising tracking.

## 2. How We Use Information

We use data to:

- Operate core app functionality (timer, history, settings)
- Restore your app state and preferences
- Support optional Together View location context
- Improve reliability and performance

## 3. Data Sharing

We do not sell personal information.
We do not share personal information with third-party advertisers.

If this changes in a future release, this policy will be updated before release.

## 4. Data Retention

- Local app data is retained on your device until you delete it or uninstall the app
- If future cloud sync features are enabled, retention details will be documented here before rollout

## 5. Permissions

Crest may request the following permissions:

- Notifications: timer completion alerts
- Location (optional): Together View geographic context

You can revoke permissions anytime in iOS Settings.

## 6. Children's Privacy

Crest is intended for general audiences.
If you are under 13, please use the app with a parent or guardian's guidance.

## 7. Your Choices

You can:

- Disable permissions in iOS Settings
- Clear app data by uninstalling the app

## 8. Changes to This Policy

We may update this policy from time to time.
When we do, we will update the "Last updated" date.
